During this project, I practiced:

* Semantic HTML structure
* Using CSS variables for reusable colors
* Flexbox centering techniques
* Creating responsive card layouts
* Styling images with border radius
* Improving code organization and readability

Example:

```css
body{
    min-height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
}
```

This helped me understand how Flexbox can be used to perfectly center content both horizontally and vertically.
